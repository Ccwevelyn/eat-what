# 今天吃啥 · 微信小程序

面向 **澳门** 发布的、**大家都可以用** 的微信小程序：通过定位 + 距离选择随机推荐附近餐厅，或通过菜系转盘决定今天吃啥。  
**餐厅与距离数据来自 OpenStreetMap（Overpass API）**，完全免费、无需 API Key、不收费。

> **不会部署？** 请直接打开项目里的 **《部署与配置步骤.md》**，里面有**第二步（部署后端）和第三步（配置小程序）的一步一步操作**，按顺序做即可。

---

## 一、你需要准备的软件与环境

### 1. 微信开发者工具（必装）

- **是什么**：腾讯官方的小程序 IDE，用来写代码、预览、调试和上传代码。
- **下载**：打开 [微信公众平台 - 小程序 - 开发 - 开发管理 - 开发设置](https://mp.weixin.qq.com/)，在页面里找到「微信开发者工具」下载链接；或直接搜索「微信开发者工具」进入官网下载。
- **安装**：下载 Windows 版，安装后用微信扫码登录即可。

### 2. 微信公众平台账号（必选）

- **是什么**：小程序的「身份」，没有账号无法发布、也无法在真机上用正式环境调试。
- **注册**：
  1. 打开 [https://mp.weixin.qq.com/](https://mp.weixin.qq.com/)
  2. 选择「立即注册」→「小程序」
  3. 按提示填写邮箱、密码、主体类型（个人即可）等，完成注册
- **重要**：注册成功后，在 **开发 → 开发管理 → 开发设置** 里可以看到 **AppID**，后面在项目里要填这个。

### 3. Node.js（发布时需用于部署后端）

- 项目自带 **server/** 后端，用 **OpenStreetMap（Overpass API）** 免费查餐厅并算距离，**无需任何 API Key、不收费**。部署后端需要本机安装 [Node.js](https://nodejs.org/)（建议 18+）。若暂时只在开发者工具里用模拟数据玩，可以不装。

## 二、用微信开发者工具打开本项目

1. 打开 **微信开发者工具**。
2. 选择 **「+」→「导入项目」**。
3. **目录**：选择你放「今天吃啥」的文件夹（即包含 `app.json`、`project.config.json` 的那一层）。
4. **AppID**：
   - 若已注册小程序：在「小程序」一栏填入你的 **AppID**（在公众平台 开发管理 → 开发设置 里）。
   - 若还没有 AppID：选 **「测试号」** 或 **「不使用 AppID」**，只能在本机模拟器里运行，不能真机预览/发布。
5. 点击 **「导入」**，即可在模拟器里看到小程序。

## 三、项目结构说明

```
EatWhat/
├── app.js              # 小程序入口逻辑
├── app.json            # 全局配置（页面路径、窗口样式、权限等）
├── app.wxss            # 全局样式
├── config.js           # 配置（API 地址、默认经纬度等）
├── project.config.json # 开发者工具项目配置
├── sitemap.json        # 小程序 sitemap 配置
├── utils/
│   └── api.js          # 请求封装（附近/菜系餐厅）；配置了 API_BASE 则用 OSM 免费数据
├── server/              # 后端（Node.js）：Overpass API 免费查餐厅、算距离，无需 Key
│   ├── index.js        # 接口：/api/nearby、/api/cuisine
│   ├── package.json
│   └── README.md
├── pages/
│   ├── index/          # 第一页：附近随机（定位 + 0–2km + 随机一家）
│   ├── wheel/          # 第二页：菜系转盘 + 该菜系餐厅（由近到远）
│   └── about/          # 第三页：声明、鸣谢、联系邮箱
└── README.md           # 本说明
```

- **第一页（附近随机）**：申请定位 → 用进度条选 0–2km → 点「随机选一家」得到该范围内随机餐厅；数据与距离来自 **OpenStreetMap（免费）**。
- **第二页（菜系转盘）**：转盘选菜系，列出该菜系餐厅（由近到远），数据来自 OSM。
- **第三页（关于）**：数据来源说明（OSM）、非营利声明、特别鸣谢陈嘉怡、免责声明、联系邮箱。

## 四、配置说明（config.js）

打开根目录下的 `config.js`：

- **API_BASE**：你的后端接口基础地址。  
  - **不填或仍为占位**时：小程序使用模拟数据，方便本地调试。  
  - **填成你部署的后端地址**（如 `https://你的域名.com/api`）后，小程序会自动用 **OpenStreetMap** 的免费餐厅与距离数据。
- **DEFAULT_LAT / DEFAULT_LNG**：默认经纬度（已设为澳门），无定位时后端兜底使用。

## 五、在澳门发布、让大家都能用（完全免费）

- **不花钱**：项目自带的后端（`server/`）使用 **OpenStreetMap 的 Overpass API** 查餐厅，**无需 API Key、无需绑卡、完全免费**。后端根据 OSM 返回的经纬度计算与用户位置的距离（0–2km 范围一致），随机返回一家或按距离排序。
- 微信小程序端不能直接请求 Overpass，所以需要 **你自己的后端** 转发。本项目已写好后端，部署即可用。

### 发布前必做（澳门）

1. **无需申请任何地图 API Key**（OSM 免费公开使用）。

2. **部署后端（server/）**  
   - 本机进入 `server` 目录，无需 `npm install`（使用 Node 内置 `http` 与 `fetch`）。  
   - 在服务器上直接运行 `node index.js`（或使用 PM2、Docker 等），**无需设置任何环境变量**。  
   - 确保服务可通过 **HTTPS** 访问（例如 `https://你的域名.com`）。  

3. **小程序配置**  
   - 在 `config.js` 里把 `API_BASE` 改成 `https://你的域名.com/api`。  
   - 在微信公众平台 **开发 → 开发管理 → 开发设置 → 服务器域名** 里，把该域名加入 **request 合法域名**。

4. **上传与发布**  
   - 在微信开发者工具中上传代码，到公众平台提交审核，通过后发布，大家即可在澳门（或其它地区）搜索「今天吃啥」使用。

## 八、常见问题

- **模拟器里点「随机选一家」没反应？**  
  请先在第一页点「开启定位」，在模拟器里可模拟定位（开发者工具右上角「…」→ 调试 → 模拟位置）。

- **没有 AppID 能运行吗？**  
  可以选「测试号」或「不使用 AppID」在本地模拟器运行，但不能真机预览和发布。

- **想加更多菜系？**  
  在 `pages/wheel/wheel.js` 里修改 `CUISINES` 数组，并保证转盘是 8 格的话，若增加/减少数量需要同步改 `pages/wheel/wheel.wxss` 里转盘扇形的数量与角度（目前按 8 格 45° 写的）。
